Common errors and fixes

❌ "No tables found" in a Table dropdown
One of your Excel files is not formatted as a Table. Format both files using Ctrl+T before running again.
❌ No rows updated after running
The IDs in the ID Translation file don't match the IDs in the donor profile. Confirm ID column names and values are identical across both files.
❌ Expression error on contains()
Copy the expression from the Formula Code Library exactly. Column names inside expressions must match your actual column headers precisely.

Donor Outreach — Build Step 3 of 4
Optional: Skip Blank Email Rows

Why add this?

If any donor row has an empty Email cell, the Create Draft action will fail for that row. Adding this Condition tells the flow to skip blank rows gracefully.

Optional Condition — inside Apply to each, before Create Draft
Left side
Expression → empty(items('Apply_to_each')?['Email']) expression
Operator
Select is equal to · right side: type false static
If yes = run Create Draft. If no = skip silently. The False branch stays empty.
