⭐ Featured Flow — Prerequisite
Before You Build: Prepare Your Excel File
⚠️ Do this before building the flow. The "List rows present in a table" action in Power Automate can only read data formatted as an official Excel Table — not just a regular spreadsheet range. If your data is not formatted as a Table, the flow will fail to read any rows.

How to format your data as an Excel Table

1
Open your own "Clean No Frills Donor Excel Sheet"
This must be your personal copy of the file, saved to your own OneDrive or SharePoint — not a colleague's file. Power Automate will only be able to access files in your own OneDrive.
☐
2
Confirm your column headers in Row 1
Your header row should include: Full Name, Email, and Top Designation Name. Spelling and spacing must match exactly — the flow uses these names to find the right columns.
☐
3
Click any cell inside your data, then press Ctrl + T
A "Create Table" dialog will appear. Make sure "My table has headers" is checked, then click OK. Your data will now show blue striped formatting — this confirms it is an Excel Table.
☐
4
Name your Table (recommended)
Click the Table → go to the "Table Design" tab → find "Table Name" in the top-left and type a name like DonorTable. This makes it easier to find when building the flow.
☐
5
Save the file to your OneDrive
Power Automate needs the file to be stored in OneDrive for Business or SharePoint — not your local Desktop or Downloads folder.
☐

⭐ Featured Flow — Build Step 1 of 3
Create the Flow & Add the Excel Action

Create the flow in Power Automate

1
Go to flow.microsoft.com and sign in with your OSU account
Use the same credentials you use for Outlook and Teams.
☐
2
Click "+ Create" in the left sidebar → choose "Instant cloud flow"
Give your flow a name (e.g. "Donor Outreach Drafts"). Under "Choose how to trigger this flow," select "Manually trigger a flow," then click Create.
☐

Add the Excel action

Action 1 — Read your donor list
Excel Online (Business) → List rows present in a table
Click "+ New step," search for "Excel Online (Business)," and select "List rows present in a table." Then fill in these fields:
Location: OneDrive for Business
Document Library: OneDrive
File: Click the folder icon and browse to your own "Clean No Frills Donor Excel Sheet" — each person must select their own file from their own OneDrive
Table: Select the Table name you created in Excel (e.g. DonorTable). If no table appears in the dropdown, go back and confirm your Excel data is formatted as a Table first.
🚨 Use your own file — not a colleague's. Do not borrow or link to someone else's Excel file. Each team member must upload and select their own copy of the donor sheet. Your file must be in your OneDrive to appear in the file picker.
🎯
⭐ Featured Flow — Build Step 2 of 3
Add the Loop & Draft Creation

Loop through every donor row

Action 2 — Repeat for each row
Control → Apply to each
Click "+ New step," search for "Control," and select "Apply to each." This tells the flow to run the next action once for every donor in your Excel table.
Select an output from previous steps: click in the field → select value from the dynamic content panel (this comes from the "List rows" action)
Action 3 — Create a draft for each donor (inside the loop)
Office 365 Outlook → Create draft (V3)
Inside the "Apply to each" block, click "Add an action," search for "Office 365 Outlook," and select "Create draft (V3)." Fill in these fields using dynamic content:
To: click the field → dynamic content panel → select Email (from your Excel table)
Subject: type your subject line and insert Full Name as a dynamic field — e.g. "Staying Connected, [Full Name]"
Body: write your outreach message — use Full Name and Top Designation Name as dynamic fields to personalize each draft automatically
How dynamic content works: When you click inside any field (To, Subject, Body), a panel appears on the right labeled "Dynamic content." Click a column name — like Full Name or Top Designation Name — and Power Automate will insert that column's value from each Excel row automatically as the flow loops through your donor list.
🎯
⭐ Featured Flow — Build Step 3 of 3
Write Your Own Draft, Save & Run
📝 Everyone must write their own email body. Your outreach email is a personal communication from you to your donors. Each team member must write their own message in the Body field — do not copy a colleague's exact draft. A coworker's draft may serve as inspiration or a structural template, but the voice and content should be your own.

Example draft structure (for reference only — write your own)

📝 Body field — this is an example. Write your own version below it.
Dear [Full Name],

I hope this message finds you well. I'm reaching out to share some exciting updates from Ohio State and to express our sincere gratitude for your generous support of [Top Designation Name].

I would love to connect at your convenience to share how your generosity is making a real impact. Please feel free to reply to this message or reach out directly.

With appreciation,
[Your Name]
Ohio State University Advancement

Save and test your flow

1
Click "Save" in the top right — look for a green confirmation
If any red banners appear, click on them to see what needs to be fixed before saving.
☐
2
Click "Test" → "Manually" → "Run flow"
Power Automate will run your flow once. Each action should show a green checkmark when it completes successfully.
☐
3
Open Outlook and check your Drafts folder
You should see one personalized draft for each donor row in your Excel sheet — addressed to them, with their name and designation filled in.
☐
4
Review, add a personal touch if needed, and send
The hard work is already done — just review each draft and hit Send when you're ready.
☐

Flow 3 of 6
Email Response Tracker

What does this flow do?

When
A new email hits your Inbox
Triggers instantly — every time you receive an email
What it checks
Subject line match
Looks for a flagged sent email with a matching subject
If match found
Clears the flag
Removes the follow-up flag from the original sent email
And then
Moves to "Responded"
Files the sent email into a dedicated Responded folder
The big idea: Once a donor replies, your workflow cleans itself up automatically. No manual flag-clearing needed.
📥

Go to flow.microsoft.com → My Flows → confirm it shows "On"

Flag clears, email moves — all on its own
Your Sent Items stays clean without any extra work from you.

What does this flow do?

When
Every day at 5 PM
Wraps up your workday with a reply summary
What it scans
Your "Responded" folder
Looks for any emails received that day
What you get
A digest email
